# TrackingRoom

This program features the Ambience effect. Use it to add room sound without adding the decay associated with reverberation. This is a great way to add “air” to when you’re tracking direct. Stereo `EQ` (in the `Soft Row`) lets you fine tune the sparkle and boom (high and low frequencies).
